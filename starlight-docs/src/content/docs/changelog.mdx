## v0.1.2 - Major Feature Release
*April 11, 2024*

<CardGrid>
  <Card title="📊 Analytics Improvements" icon="chart">
    Enhanced DataLayer events and purchase tracking
  </Card>
  <Card title="🛒 Upsell Events" icon="upsell">
    Comprehensive upsell purchase event system
  </Card>
  <Card title="🎯 Navigation Control" icon="lock">
    Back navigation prevention on upsell pages
  </Card>
  <Card title="🎫 Voucher System" icon="voucher">
    Complete voucher system with cart updates
  </Card>
</CardGrid>

### Key Features Added:
- **DataLayer Purchase Events**: Fire on first page after order creation
- **View Item List Events**: Only fires when `os-package-id` elements exist
- **Upsell Purchase Events**: Trigger on every page after upsell acceptance
- **Custom Upsell Event**: `os_accepted_upsell` DataLayer event
- **Navigation Prevention**: Prevents back navigation on upsell pages
- **Enhanced Voucher System**: Updates entire cart automatically
