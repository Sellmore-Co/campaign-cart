## Method 1: CDN Installation (Recommended)

The fastest way to get started - just add one script tag:

<Tabs>
  <TabItem label="Production (Minified)">
    <Code code={`<script src="https://rtc2.29next.com/campaign-cart/29next.min.js"></script>`} lang="html" />
    
    **Use for:** Live websites, production environments
    
    **Benefits:** Fastest loading, CDN cached, auto-updates
  </TabItem>
  
  <TabItem label="Development (Full)">
    <Code code={`<script src="https://rtc2.29next.com/campaign-cart/29next.js"></script>`} lang="html" />
    
    **Use for:** Development, debugging, learning
    
    **Benefits:** Readable code, detailed error messages, source maps
  </TabItem>
</Tabs>

<Aside type="tip">
**CDN Benefits:** Global CDN delivery, automatic updates, browser caching, and no hosting required. This is the recommended method for most users.
</Aside>

## Method 2: NPM Installation

For modern build systems (Webpack, Vite, Parcel) and frameworks (React, Vue, Svelte):

<Steps>
1. **Install the package**
   
   <Code code={`npm install @29next/campaign-cart`} lang="bash" />

2. **Import in your JavaScript**
   
   <Code code={`import { TwentyNineNext } from '@29next/campaign-cart';`} lang="javascript" />

3. **Initialize the SDK**
   
   <Code code={`const cart = new TwentyNineNext({
     apiKey: 'YOUR_API_KEY',
     campaignId: 'YOUR_CAMPAIGN_ID'
   });`} lang="javascript" />
</Steps>

## Method 3: Direct Download

Download and host the files yourself:

<Steps>
1. **Download the latest release**
   - Visit [GitHub Releases](https://github.com/29next/campaign-cart/releases)
   - Download `campaign-cart-latest.zip`

2. **Extract and upload**
   - Extract the files to your web server
   - Upload to `/js/` or `/assets/` directory

3. **Include in your HTML**
   <Code code={`<script src="/js/29next.min.js"></script>`} lang="html" />
</Steps>

<Aside type="note">
**Self-hosting:** You'll need to manually update when new versions are released. Consider using CDN for automatic updates.
</Aside>

## Configuration Options

### Meta Tag Configuration (Simplest)

<Code 
  code={`<!-- Required -->

<Code 
  code={`window.osConfig = {

## Common Installation Issues

<CardGrid>
  <Card title="🚫 Script Not Loading" icon="error">
    **Symptoms:** Nothing happens, `window.twentyNineNext` is undefined
    
    **Solutions:**
    - Check network tab for 404 errors
    - Verify script URL is correct
    - Check for ad blockers
  </Card>
  
  <Card title="⚠️ API Key Errors" icon="warning">
    **Symptoms:** Cart loads but products don't work
    
    **Solutions:**
    - Verify API key is correct
    - Check campaign ID exists
    - Ensure API key has permissions
  </Card>
  
  <Card title="🔧 Configuration Issues" icon="wrench">
    **Symptoms:** Features missing or not working
    
    **Solutions:**
    - Check meta tag syntax
    - Verify campaign settings
    - Enable debug mode for details
  </Card>
</CardGrid>
