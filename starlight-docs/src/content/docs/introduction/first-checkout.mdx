### Key Features Explained

<Tabs>
  <TabItem label="Product Selector">
    **`data-os-component="selector"`** creates a group where only one item can be selected at a time. Perfect for main product choices like Basic vs Premium.
  </TabItem>
  
  <TabItem label="Add-on Toggles">
    **`data-os-action="toggle-item"`** creates independent checkboxes for optional add-ons that customers can include or skip.
  </TabItem>
  
  <TabItem label="Live Cart Updates">
    Elements with **`data-os-cart-*`** attributes automatically update when customers change their selection - no custom JavaScript needed!
  </TabItem>
</Tabs>

## Testing Your Complete Flow

<Steps>
1. **Start on Product Page** - Select a package and add-ons
2. **Go to Checkout** - Fill out the form with test information
3. **Use Test Payment** - Card number: `4111 1111 1111 1111`, any future date, any CVV
4. **Complete Purchase** - You should reach the receipt page
5. **Verify Receipt** - All order details should display correctly
</Steps>

<Aside type="tip">
**Test Information:**
- **Email:** Use a real email to receive confirmation
- **Card:** `4111 1111 1111 1111` (Visa test card)
- **CVV:** Any 3-digit number
- **Expiry:** Any future date
</Aside>
