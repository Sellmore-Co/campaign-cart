## What is Campaign Cart?

A JavaScript SDK that connects the 29next API to web pages through HTML attributes. Works with Webflow, custom HTML, or any page builder.

<Aside type="tip">
Campaign Cart is designed to be framework-agnostic and works with any HTML page or page builder, making integration simple regardless of your tech stack.
</Aside>

## How it works

<Steps>
1. **Include the script** - Add the Campaign Cart JavaScript to your page
2. **Add meta tags** - Configure API keys and campaign settings  
3. **Use HTML attributes** - Add data attributes to buttons and elements
4. **Automatic handling** - SDK manages all API calls and interactions
</Steps>

## Architecture

```
HTML Page (data-os-* attributes)
         ↓
Campaign Cart JavaScript
         ↓
29next Campaign API
```
