## Turn Any Website Into a Complete Checkout System

Campaign Cart is a **JavaScript SDK** that adds shopping cart, checkout, and payment processing to any website in minutes. Perfect for **junior developers** and **no-code builders** who need e-commerce functionality without complex integrations.

<Aside type="tip">
**No backend required!** Campaign Cart handles all the complex e-commerce logic, payment processing, and order management through the 29next API.
</Aside>

## How it works

<Steps>
1. **Include the script** - Add the Campaign Cart JavaScript to your page
2. **Add meta tags** - Configure API keys and campaign settings  
3. **Use HTML attributes** - Add data attributes to buttons and elements
4. **Automatic handling** - SDK manages all API calls and interactions
</Steps>

## Architecture

```
HTML Page (data-os-* attributes)
         ↓
Campaign Cart JavaScript
         ↓
29next Campaign API
```
