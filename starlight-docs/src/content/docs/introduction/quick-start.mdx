Get Campaign Cart working on your website in **5 minutes**. This guide shows you the absolute fastest way to add shopping cart functionality to any HTML page.

## What You'll Build

By the end of this guide, you'll have:
- ✅ A working "Add to Cart" button
- ✅ Live cart count and total display
- ✅ A functional checkout button
- ✅ Real payment processing (test mode)

<Aside type="tip">
**Before you start:** You'll need your API credentials from the 29next dashboard. Don't have them? [Sign up for free](https://29next.com) to get your API key and campaign ID.
</Aside>

## Step 1: Add the Script

Add this single line to your HTML `<head>` section:

<Code 
  code={`<script src="https://rtc2.29next.com/campaign-cart/29next.min.js"></script>`}
  lang="html" 
  title="Add to your <head> section"
/>

## Step 2: Configure Your Credentials

Add your API credentials using meta tags (replace with your actual values):

<Code 
  code={`<meta name="os-api-key" content="YOUR_API_KEY">
<meta name="os-campaign-id" content="YOUR_CAMPAIGN_ID">`}
  lang="html" 
  title="Add to your <head> section"
/>

<Aside type="caution">
**Replace the placeholder values!** Get your actual API key and campaign ID from your [29next dashboard](https://dashboard.29next.com).
</Aside>

## Step 3: Add Cart Functionality

Add these elements anywhere in your `<body>`:

<Code 
  code={`<!-- Product Button -->
<button data-os-action="toggle-item" data-os-package="1">
  Add Starter Kit - $97
</button>

<!-- Cart Display -->
<div>
  Items in cart: <span data-os-cart-count>0</span>
  Total: <span data-os-cart-total>$0.00</span>
</div>

<!-- Checkout Button -->
<button os-checkout-payment="combo">
  Proceed to Checkout
</button>`}
  lang="html" 
  title="Add to your <body> section"
/>

## Step 4: Test It!

<Steps>
1. **Open your page** in a web browser
2. **Click "Add Starter Kit"** - the cart count should change to "1"
3. **Check the total** - it should show the product price
4. **Click "Proceed to Checkout"** - a checkout form should appear
5. **Use test card** `4111 1111 1111 1111` to complete a test purchase
</Steps>

<Aside type="tip">
**It's working if:** The cart count updates when you click the button and the checkout form appears when you click checkout. If not, check the browser console for error messages.
</Aside>

## Common Issues & Solutions

<CardGrid>
  <Card title="🚫 Nothing Happens" icon="warning">
    **Check:** 
    - API credentials are correct
    - Script loaded (no 404 in console)
    - No JavaScript errors in console
  </Card>
  
  <Card title="💸 Cart Total Shows $0" icon="error">
    **Check:**
    - Package ID exists in your campaign
    - Campaign is active
    - API key has correct permissions
  </Card>
  
  <Card title="🔄 Checkout Won't Open" icon="x">
    **Check:**
    - Campaign has payment methods enabled
    - Page meta tags include campaign ID
    - No ad blockers interfering
  </Card>
</CardGrid>
